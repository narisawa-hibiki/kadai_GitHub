# ASerial_lib_Win
## 概要
ASerial_lib_Controller_Winは、Windows環境でのシリアル通信を簡単に行うためのC++ライブラリです。
このライブラリは、ASerialCoreをベースにしており、シリアル通信の基本的な機能を提供します。

## 開発環境
- C++14以上
- Windows環境

## 開発者
- cha-maru(@tappei2112)

## ディレクトリ構成
- ASerialCore : ASerialのコアライブラリ
- /test : テスト用のサンプルコードディレクトリ
- /WindowsSerial : Windows環境でのシリアル通信を行うためのライブラリディレクトリ
- ASerial_lib_Controller_Win.h : ASerial_lib_Winのヘッダファイル
- ASerial_lib_Controller_Win.cpp : ASerial_lib_Winの実装ファイル
- LICENSE.txt : ライセンスファイル
- README.md : このドキュメントファイル
- CHANGELOG.md : チェンジログファイル

## 利用方法
- ASerial_lib_Winフォルダを自身のプロジェクトディレクトリに追加して下さい。
- プログラムファイルで`ASerial_lib_Controller_Win.h`をインクルードしてください。
